Adds a sidebar widget that display a heating calculator.


== Description ==

This plugin adds a sidebar widget that displays a heating calculator, you input the details of the location that you require heating for and the calculator recommends the approptiate heating unit(s).

= Version 1.0 =
First release.


== Installation ==

1. Unpack the *.zip file and extract the /heating-calc/ folder and the files.
2. Using an FTP program, upload the full /heating-calc/ folder to your WordPress plugins directory (Example: /wp-content/plugins).
3. Go to Plugins > Installed and activate the plugin.
4. Go to Appearance > Widgets, add the widget to a sidebar and save.


== Frequently Asked Questions ==

= The plugin is great but can I change the colour to fix in better with my WordPress theme? =
Yes you can. Simply change the background image `heating-calc/images/heating-calc.jpg` and the colour of `container` in `heating-calc/styles/heating-calc.css`.
